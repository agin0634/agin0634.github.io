[PYTHON/SELENIUM] 使用SELENIUM自動化登入

Created @Jun 4, 2020 6:39 PM
Tags Python
最近因為某些原因接觸到網路爬蟲，碰到了登入的問題所以記錄在這篇文章，這邊用Facebook作範例。

安裝Selenium

首先，安裝Selenium。沒甚麼問題，一行搞定 

pip install selenium

安裝WebDriver

Selenium和瀏覽器溝通需要安裝WebDriver，我這邊使用Chrome

將解壓縮的exe檔放在python安裝資料夾或.py的資料夾

瀏覽器開啟Facebook

先簡單的測試能不能打開Facebook 

from selenium import webdriver

url = 'https://zh-tw.facebook.com/'

driver = webdriver.Chrome()
driver.get(url)

成功!現在開始尋找元素定位

尋找元素定位

尋找元素定位的方法很多，選一個適合的就好

這邊我使用 id 來尋找 

driver.find_element_by_id()

回到瀏覽器，按下F12找到email input box的id，password同理

再使用 send_keys() 填入自己的帳號密碼 

driver.find_element_by_id('email').send_keys(emailaddress)
driver.find_element_by_id('pass').send_keys(password)

接著，找到login按鈕的id，使用 click() 模擬滑鼠點擊 

driver.find_element_by_id('loginbutton').click()

完整代碼: 

from selenium import webdriver

url = 'https://zh-tw.facebook.com/'

driver = webdriver.Chrome()
driver.get(url)

emailaddress = '你的信箱'
password = '你的密碼'

driver.find_element_by_id('email').send_keys(emailaddress)
driver.find_element_by_id('pass').send_keys(password)
driver.find_element_by_id('loginbutton').click()
